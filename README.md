## 💻 Installation & Setup

1. Clone the repository:
```bash
git clone https://github.com/lannqt/technical-test.git
cd technical-test
```

2. Install Dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```
4. Open in your browser:
```bash
http://localhost:5173
```
